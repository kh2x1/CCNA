# أكاديمية الشبكات — شرح CCNA و Network+

موقع تعليمي عربي (ثابت، بدون أي اعتماديات) يشرح أهم مواضيع شهادتي
**CCNA (200-301)** و **CompTIA Network+** بأسلوب مبسّط مع أمثلة عملية وأوامر Cisco.

## المحتوى

- **البداية والأساسيات**: تعريف الشبكات، الفرق بين الشهادتين، خطة المذاكرة.
- **نماذج الشبكات**: OSI، TCP/IP، المنافذ والبروتوكولات.
- **العنونة و Subnetting**: IPv4، تقسيم الشبكات، VLSM، IPv6.
- **التبديل Switching**: عمل السويتش، VLANs، STP.
- **التوجيه Routing**: المسارات الثابتة، OSPF، جدول التوجيه.
- **خدمات الشبكة**: DHCP، DNS، NAT/PAT.
- **الأمان**: مبادئ CIA، تأمين الأجهزة، ACLs، Port Security.
- **اللاسلكي والوسائط**: Wi-Fi، الكابلات والألياف.
- **العمليات والأتمتة**: NTP/Syslog/SNMP، SDN/REST/JSON، استكشاف الأخطاء.

## التشغيل

افتح `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا:

```bash
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

## البنية

```
index.html      # الهيكل الرئيسي
css/style.css   # التنسيق (مع وضع ليلي ودعم RTL)
js/data.js      # محتوى الدروس
js/app.js       # التنقّل والبحث والتوجيه
```

أُنشئ بهدف تعليمي.
